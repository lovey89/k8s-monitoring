** Setup this repo

#+BEGIN_SRC bash
helm repo add prometheus-community https://prometheus-community.github.io/helm-charts
helm repo update
helm pull prometheus-community/kube-prometheus-stack
tar -xzf kube-prometheus-stack-*.tgz
rm kube-prometheus-stack-*.tgz
#+END_SRC
